圣杯布局的概念

即一种轻便，广泛使用的三栏流体布局。它具有以下的优秀特性：

  1. 两边固定宽度而中间可以流动；
  2. 允许中间栏代码最先出现；
  3. 允许任一栏的高度都最高；
  4. 仅需一个额外的div标签；
  5. 仅需很少必要css代码及最少兼容性补丁代码。
